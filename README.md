**lita-poop** is a handler for [Lita](https://github.com/jimmycuadra/lita) that provides a random euphemism for going #2.

## Installation

Add lita-poop to your Lita instance's Gemfile:

``` ruby
gem "lita-poop"
```

## Usage
 
    Lita: poop
